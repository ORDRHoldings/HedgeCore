Design System — Unified Tokens + Shared Components

COLOR SYSTEM
Sidebar / Deep background
Page background
Panel / Card
Accent — active states, CTAs ONLY
Primary text
Secondary text
Tertiary / disabled
Status colors (green/red/amber) appear ONLY on data values — never on UI chrome, badges, or buttons.
SHARED COMPONENTS
  • PageHeader — icon + title + breadcrumb + action slot
  • PageShell — wraps every page (header + content area)
  • KpiStrip — horizontal stat bar (reusable across pages)
  • DataTable — standard table with sort/filter header
  • EmptyState — loading / error / no-data (already exists, enforce usage)
  • ActionButton — primary (blue fill) / secondary (border) / ghost
  • StatusDot — green/red/amber dot for data status
  • tokens.ts — single source of truth, replaces 5 local S/T objects
TYPOGRAPHY RULES
Page Titles
IBM Plex Sans · 20px · 700
Section Headers
IBM Plex Sans · 14px · 500
Data Values
IBM Plex Mono · 13px · 400
Minimum font size: 12px everywhere. No exceptions.
