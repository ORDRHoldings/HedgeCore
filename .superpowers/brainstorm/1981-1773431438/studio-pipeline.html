Report Studio: 4-Step Pipeline

The Studio tab guides users through report creation. Which workflow feels right?

A

Linear Pipeline (Hedge Desk style)

Strict step-by-step. Must complete each step before advancing. "STEP 1 OF 4" header bar.

1 BIND
→
2 COMPOSE
→
3 PREVIEW
→
4 EXPORT
Best for: New users, compliance workflows where order matters
Risk: Power users find it slow for repeat exports
