Forward fix: `151c591` â€” replaced `SET LOCAL <var> = :param` with `SELECT set_config('<var>', :param, true)`. Same transaction-local semantics, but `set_config()` is a function call so the bind protocol works. Health restored 17:28Z (~4 min after push).

Post-mortem: `docs/incidents/2026-05-16-rls-set-local-bind-params.md`. Two new HIGH risks opened (RISK-CI-PG-01: no PG tests in CI; RISK-OPS-MON-01: no 5xx alert, no auto-rollback â€” both directly responsible for the 3-day silent degradation).
