# ORDR Terminal Design System

Portable UI/UX design system for institutional-grade financial applications.

## Features

- 7 theme presets (5 dark, 2 light)
- 8 curated accent colors with contrast validation
- 6 operational templates for different workflows
- WCAG 2.1 AA contrast checking
- 3-level density scaling (compact/standard/spacious)
- Configurable typography (3 UI fonts, 3 mono fonts, 5 sizes)
- Accessibility: reduced motion, high contrast, color+icon modes
- 11 production-ready UI components
- CSS variable architecture for instant theme switching

## See CLAUDE.md for integration instructions.

## Tech Stack

- React 18+ / Next.js 14+
- TypeScript 5+
- CSS Variables (no CSS-in-JS runtime)
- lucide-react for iconography
- IBM Plex Sans / Mono, Inter, JetBrains Mono, Manrope
